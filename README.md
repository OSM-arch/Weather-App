<h1 align="center">🌦️ Weather Application</h1>

<div align="center">
  <strong>A responsive weather app using OpenWeatherMap API</strong>
</div>

<br>

<h2>🚀 Features</h2>
<ul>
  <li><b>🔍 City search</b> - Get weather for any location</li>
  <li><b>📅 Date display</b> - Formatted as "Weekday, Month Day, Year"</li>
  <li><b>🌡️ Temperature</b> - In Celsius with "feels like"</li>
  <li><b>☁️ Dynamic icons</b> - Changes based on weather conditions</li>
  <li><b>💧 Humidity</b> - Current humidity percentage</li>
</ul>

<h2>⚙️ Installation</h2>
<pre><code>git clone https://github.com/yourusername/weather-app.git
cd weather-app</code></pre>

<h2>🔑 API Setup</h2>
<ol>
  <li>Get free API key from <a href="https://openweathermap.org/api">OpenWeatherMap</a></li>
  <li>Replace in <code>script.js</code>:
<pre><code>const APIkey = 'your_api_key_here';</code></pre>
  </li>
</ol>
